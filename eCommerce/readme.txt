I have created an online dress boutique taking the inspiration from the Tueta Matoshi fashion line.
CSS Grid: It is used to style the different categories of dresses
CSS flexbox: It is used to style the navigation bar of the webpage
Variables: The variables are used to add values to colours and fonts
Nesting: Many classes and tags are used to nest the different CSS
Interpolation: It is used in the buttons scss file
Custom Properties: Main background colour is used to set with this
Functions: It is used to change the colour as per the input
Loops: loops the used to give the spaces between the paragraphs
Mixin: It is used to make use of one class properties in another
PlaceHolder selectors: The placeholder is used in buttons and gallery scss file
